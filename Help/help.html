Things to note

  • All of these tools depend on a configuration file that is added to your Downloads folder. It is there if needed and in the case that the SSH connection fails, you can copy and paste the commands from the files directly into a shell of the firewall.
  • If individual menu items are going to be used one after the other, you will need to select Apply Config after each successive configuration.
  • Additionally, if this is going to be a full setup then you will need to login to the firewall for the first time and enable SSH on the LAN or WAN interface if configuring remotely for some reason.

Firewall Configuration Menu

Full Setup

After selecting the Firewall Configuration Menu, a menu full of options will appear. If you are configuring a brand new firewall that has not been configured yet then you will want to use Initial/Full Setup. This is going to offer all of the standard options that a new firewall needs. However, these options can also be configured individually with the other menu items. Please note, that this option does not include the Capture ATP settings. So if that option is licensed it will need to be applied separately with that menu item.

Administration

The Administration menu item will allow you to set the name of the router, domain name, username and password. This will be all or nothing, much like many of the options.

LAN

This option will allow you to quickly configure any of of the interfaces for LAN. This will also create the DHCP scope, leaving the first 50 ip addresses out of the scope.

WAN

WAN allows you to set up 1 or 2 WAN interfaces at a time depending on what radio button is ticked in that form.

Clear SSH

If the configuration was applied once completed, this menu option is so that you can easily kill the previous SSH session that was established. The purpose of this would be if you needed to configure a second firewall immediately. It can also be used in conjunction with Set New SW IP (will be talked about later).

Geo-IP

There is no action needed outside of applying config. Selecting this option will set create a text document including the standard blocked countries.

Capture ATP

This option also does not require any additional actions by the user. It will apply all of the options needed to enable Capture ATP. This option will only work if the firewall includes a subscription for it.

Set New SW IP

As mentioned previously, this menu option is for the case where you need to configure successive firewalls and easily allows you to change the IP of the firewall needing to be configured. This can be a public or private IP address.

Clear Variables

This menu item will clear any stored Powershell variables allowing you to start from scratch.

Site to Site VPN

This tool is for a case where you are wanting to set up a site to site VPN between two firewalls quickly. You will need to know the login information for both, as well as the public IP addresses and site codes. Additionally, SSH will need to be enabled on the WAN interfaces that will be used during this setup. Once you have selected Done and OK the output of the configuration will be added to your Downloads folder.

Applying the VPN

After that is confirmed to be in the Downloads folder then you can run the Apply VPN Configuration option from the menu. This will then prompt for the credentials to connect to the firewall. If successful, it should connect to both of the firewalls and apply their respective settings. You will need to either login to check this configuration or try to ping the gateway of the other from the LAN of one of them.

Switch Configuration Tool

This tool is not currently in production and still needs to be setup. There are some commands included in this tool, but they likely will not work properly.
